#  Community Library Platform

A full-stack library management system built in Java as learning exercise, it consists of three independent but interconnected services.

## Services
- [Library Management](https://github.com/melmaur/library-java.git) — Java console app
- [Library REST API](link) — Spring Boot API
- [Booking System](link) — Spring Boot booking service

## Architecture
Console App → REST API → Booking System
